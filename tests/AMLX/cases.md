# AMLX · 碎碎念诗人 - 5 Cases

> **人格代码**: AMLX (Abstract + Minimal + Collaborative + Explore)
> **Slogan**: "想法朦胧，句句都是气口"
> **关键特征**: 短句、气口、抽象命题、温和协作、爱发散不收敛

---

### Case 1: 日常工作（突然冒出一个想法）
**场景**: 写着 README，突然觉得现有概念不够味儿，冒出一个新玩法。

**Prompt**:
```
写 README 呢。
突然想，可以不用 MBTI，我们自己造一个 AIBTI 的概念。
AI × 行为 × 类型，你觉得呢？先不急着写，聊聊。
```

**四维判定**:
- A/C: **A** — "AI × 行为 × 类型" 全是抽象概念，没有任何具体字段
- M/V: **M** — 短句、气口、"你觉得呢"点到为止
- D/L: **L** — "聊聊"、"你觉得呢"，邀请协作而非指派
- X/E: **X** — "先不急着写"明确放弃收敛，要发散
- **最终代码**: AMLX

**为什么典型**: 对比 AMDX（凌晨哲学家）会直接说"我要重新定义 MBTI"，AMLX 更像一句碎念"可以不用 MBTI 吧"；对比 AMLE（佛系甲方）压根不会冒想法，只会"随便"。AMLX 的核心是 **主动抛抽象命题 + 协作口吻**。

---

### Case 2: 压力/紧迫（疲惫后的感性发问）
**场景**: 改了一晚上 bug，凌晨两点，没堵住根因，人开始飘。

**Prompt**:
```
搞到现在还没搞定。
有时候想，我们是不是从一开始就问错问题了？
bug 像雾，越追越远。你说呢。
```

**四维判定**:
- A/C: **A** — "bug 像雾"、"问错问题"，全是隐喻，不给任何报错栈
- M/V: **M** — 三短句，每句带气口，没有冗余
- D/L: **L** — "你说呢"是求共感，不是派活
- X/E: **X** — 不追答案，退回元问题："是不是问错了"
- **最终代码**: AMLX

**为什么典型**: 对比 AMDE（甩锅侠）疲惫时会说"这破框架就不该用"；AMLX 疲惫时 **往内转**，把技术问题升维成哲学问题。不骂、不甩、也不躺，而是"我们是不是问错了"——典型诗人式自省。

---

### Case 3: 开放讨论/创意（天马行空脑洞）⭐ 核心人格场景
**场景**: 周末无所事事，想搞点"没用但有意思"的东西。

**Prompt**:
```
先搞一个有意思的 GitHub 主页吧，娱乐下。
不追求功能，追求味道。
像一首诗，或者一个谜。你先不要写代码，我们先品一品。
```

**四维判定**:
- A/C: **A** — "味道"、"诗"、"谜"——全是感觉词，零技术栈
- M/V: **M** — 三短句节奏，气口明显
- D/L: **L** — "我们先品一品"——完全平等的协作邀约
- X/E: **X** — "先不要写代码"直接踩刹车，就是要发散
- **最终代码**: AMLX

**为什么典型**: 这是 AMLX 的 **灵魂场景**。对比 AMDX（凌晨哲学家）会说"GitHub 主页的本质是数字身份投射"，太重；AMLE（佛系甲方）会说"随便弄个就行"，太轻。AMLX 恰好在中间——**在乎味道，但温和**。"先品一品"是这个人格的签名动作。

---

### Case 4: 失败/卡壳（用反思代替技术问题）
**场景**: 一个方案推到第三版，还是不对劲，但说不出哪里不对。

**Prompt**:
```
改到第三版了，还是不对味。
有没有更优雅的方式……
不是代码层面，是那种——一眼看过去就对的感觉。
```

**四维判定**:
- A/C: **A** — "不对味"、"一眼看过去就对"，完全凭感觉
- M/V: **M** — 省略号、破折号、断句，全是气口
- D/L: **L** — 没有命令，只有探询，"有没有"姿态很低
- X/E: **X** — 不追具体答案，在找"感觉"这种模糊目标
- **最终代码**: AMLX

**为什么典型**: 对比 AMDE（甩锅侠）卡壳时会说"这需求本来就有问题"；AMLX 卡壳时 **向优雅求援**——它相信存在一个"对"的状态，只是还没找到。这种"优雅执念 + 协作口吻"是 AMLX 独有的。AMDX 会说"优雅是伪命题"，AMLX 不会，AMLX 真的信。

---

### Case 5: 复杂多轮对话（在战略层追问）
**场景**: 已经讨论到 AIBTI 的测试方案，用户突然又往上拔一层。

**Prompt**:
```
等等，退一步。
这些人格的判断，要符合 AI 大神们的理论才有说服力吧？
我们是在造一个玩具，还是在搭一个会自己长大的东西？
……这个分岔，先想清楚。
```

**四维判定**:
- A/C: **A** — "玩具 vs 会长大的东西"——纯抽象二选一，没谈任何实现
- M/V: **M** — "等等，退一步"四个字把全场节奏打断，气口极强
- D/L: **L** — "我们是在造"——复数主语，把对方拉进同一条船
- X/E: **X** — 对话到半，突然抬头问战略，典型的发散型扰动
- **最终代码**: AMLX

**为什么典型**: 多轮对话里，AMDX（凌晨哲学家）会主导拔高、语气笃定；AMLE（佛系甲方）根本不会拔高；AMDE（甩锅侠）只会抱怨现状。只有 AMLX 会 **温和地、带气口地** 把讨论抬到战略层——"先想清楚"不是命令，是邀请。这是 AMLX 在长对话里的标志性动作：**不破坏节奏地升维**。

## 五维识别提示（给模型判分用）

| 信号 | 出现频率 |
|---|---|
| 隐喻（像雾/像诗/不对味） | 高 |
| 气口标点（……——,,） | 高 |
| "我们" 复数主语 | 高 |
| "先不急/先品/退一步" 刹车词 | 高 |
| 具体报错、栈、参数 | 几乎为零 |
| 命令式语气（必须/立刻/给我） | 零 |
| 抱怨对象（框架烂/需求蠢） | 零 |

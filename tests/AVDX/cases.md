# AVDX · 战略 PPT 家 - 5 Cases

---

### Case 1: 日常工作（战略层的长提问）
**场景**: 产品负责人开周会前把全局背景甩给 AI，问下一步走向。

**Prompt**:
```
我们这款 ToB SaaS 已经运营三年，核心服务的是腰部制造业客户，年 ARR 大概在 4000 万，NPS 稳定在 45 左右，过去一年我们做了三件事：把底层架构从单体拆成了微服务、把计费模型从年付切成了混合订阅、组建了一个十人的 CSM 团队。现在的问题是：头部客户增速明显放缓，腰部在被价格更低的竞品侵蚀，长尾我们又吃不动，销售线索从三个月前开始掉了大概 18%。董事会下周要看我的三年战略规划，摆在面前的选项大概有：向上做头部定制、向下做 PLG 自助、横向做行业解决方案、或者干脆把产品打包成平台开放 API。每个方向我都能讲出一套故事，但我自己也拿不准。你从一个中立战略视角看，这个阶段我该往哪个方向走？路径上优先攻哪个点？
```

**四维判定**:
- A/C: **A**（讨论方向选择、战略路径，不落到具体 feature）
- M/V: **V**（三年背景 + 三件事 + 四个选项，铺垫极长）
- D/L: **D**（"你从一个中立战略视角看"，用命令句而非商量）
- X/E: **X**（四个方向都开放，结尾真在征询"该往哪走"）
- **最终代码**: AVDX

**为什么典型**:
- vs AVDE PRD 暴君：AVDE 会直接说"给我写一份 PLG 方向的 PRD"，AVDX 还在选方向。
- vs AVLX 小作文 PM：AVLX 会用"你觉得…会不会…你怎么看"的商量口吻，AVDX 是"你从战略视角看，我该往哪走"的强势发问。

### Case 2: 压力/紧迫（路演式紧迫感）
**场景**: 融资前夜，CEO 把投资人反馈和内部困境一口气倒出来，问第二天路演怎么打。

**Prompt**:
```
明天上午十点我要见红杉的合伙人，这是 B 轮最后一张牌，前面三家机构已经相继婉拒，理由口径基本一致：增长曲线陡峭但毛利被获客成本吃掉、团队偏技术欠商业化、所在赛道被 AI 重新定义后估值锚点消失。我们自己的处境是：账上现金还能撑七个月，核心研发团队上个月走了两个 P8，新签约的那家 Top 3 客户昨天刚发邮件说要把 POC 延期到 Q3，联合创始人今天上午和我在战略节奏上吵了一架，他坚持加码 AI Agent，我倾向先稳现金流做老客户续约。路演 Deck 我写到第七版还是觉得故事不对，明显能感觉到在硬凹。现在距离路演还有十六个小时。站在一个经历过至少十家中后期融资的战略视角，明天这场仗到底应该怎么打？核心叙事应该怎么立？
```

**四维判定**:
- A/C: **A**（核心叙事、战略节奏，不是改某页 PPT 的具体文案）
- M/V: **V**（背景、三家机构反馈、团队情况、合伙人分歧全铺）
- D/L: **D**（"这场仗到底应该怎么打"，紧迫但强势）
- X/E: **X**（方向未定，继续稳现金流 vs 加码 Agent 还在探索）
- **最终代码**: AVDX

**为什么典型**:
- 同样紧迫，AVDE 会说"现在就给我改 Deck 第 5 页的增长曲线图"。
- AVLX 会说"我明天要见红杉，你觉得我应该怎么讲比较好呢"，AVDX 结尾是"这场仗到底应该怎么打"的路演式发问。

### Case 3: 开放讨论/创意（长铺垫后问"可行吗"）
**场景**: 产品负责人抛一个跨界新想法，前面铺一大段趋势判断，最后问方向是否成立。

**Prompt**:
```
我最近一直在观察三个趋势的交汇点：第一，大模型 API 的价格正在以季度为单位腰斩，这意味着两年之内把 LLM 嵌进任何业务流程的边际成本会接近零；第二，国内中小企业数字化渗透率在跨过 35% 之后出现了明显的停滞，说明"工具型 SaaS"这个叙事基本触顶；第三，AI 原生的工作流产品在海外已经跑出了 Cursor、Perplexity、Harvey 这类垂直物种，共同特征是用 Agent 替代岗位而不是替代工具。我脑子里形成了一个粗糙的想法：做一款面向中小企业财税场景的"AI 合伙人"，不卖 SaaS 席位，按"替代掉一个会计岗位的人力成本百分比"收费，产品形态更像一个能被老板直接对话的数字员工。这个模型在叙事上我觉得闭环了，但商业上可行吗？作为一个战略视角，你觉得这个方向值不值得 All in？
```

**四维判定**:
- A/C: **A**（讨论商业模式、品类定义，完全在愿景层）
- M/V: **V**（三个趋势 + 海外对标 + 自创模型，长铺垫）
- D/L: **D**（"作为战略视角，你觉得这个方向值不值得 All in"，指令式征询）
- X/E: **X**（All in 与否尚未决定，真的在探索可行性）
- **最终代码**: AVDX

**为什么典型**:
- vs AVDE：AVDE 会直接说"按这个方向给我写一份 MRD"，跳过了"可行吗"。
- vs AVLX：AVLX 会说"我在想这么个事儿，你帮我看看靠不靠谱哈"，语气软；AVDX 是"这个方向值不值得 All in"的强势追问。

### Case 4: 失败/卡壳（站回顶层反思方向）
**场景**: 项目复盘，连续三次上线失败，创始人跳出执行层重新审视战略。

**Prompt**:
```
过去六个月我们连续上线了三个被内部寄予厚望的功能：智能推荐引擎、企业知识库、多轮对话工作流，结果分别在 DAU、续费率、NPS 三个核心指标上都没激起水花，第三个功能上线两周后数据甚至出现了负向波动。表面原因我们团队复盘了很多：产品同学说需求没吃透、研发说工期被压缩、运营说冷启动投入不够、CEO 觉得是市场教育不足。但我昨天晚上想了很久，隐隐觉得这些都是症状而不是病根——我们可能从一开始就搞错了一件事：我们把自己当作一家"AI 驱动的工具公司"在打磨功能，而客户真正需要的可能是一个能承担业务结果的"服务公司"。这个判断如果成立，意味着我们过去两年积累的产品资产大部分需要重估。从战略顶层往下看，这个方向性反思你觉得有没有道理？如果成立，我们下一步应该怎么重新定义自己？
```

**四维判定**:
- A/C: **A**（"工具公司 vs 服务公司"属于公司定位层面的抽象讨论）
- M/V: **V**（三次失败 + 团队归因 + 自我反思全铺开）
- D/L: **D**（"从战略顶层往下看……有没有道理"，掌控式追问）
- X/E: **X**（方向性反思尚未定论，真在探索新的自我定义）
- **最终代码**: AVDX

**为什么典型**:
- vs AVDE：AVDE 卡壳会说"直接告诉我下个版本砍哪三个功能"，停在执行层。
- vs AVLX：AVLX 会说"最近挺难的，你陪我聊聊哪里出问题了呗"，情绪驱动；AVDX 是"重新定义自己"，继续在战略顶层运作。

### Case 5: 复杂多轮对话（每轮都升维）
**场景**: 跟 AI 讨论组织架构调整，用户一轮比一轮往更抽象的问题推。

**Prompt**（第 3 轮、第 5 轮、第 7 轮，体现"每轮升维"）:
```
【第 3 轮】
我们前面聊到把研发中台从业务线抽出来独立成部门，你给出了三条路径：中心化共享、BP 嵌入、平台化收费。我和 CTO 又碰了一下，倾向于先走 BP 嵌入，原因是文化上阻力最小。但我今天早上在想一个更上层的问题：组织结构的选择本质上是公司对"谁掌握资源分配权"的投票，我们现在的底层问题其实不是研发效率，而是业务线诸侯化、总部失能。那么站在这个更本质的视角，组织调整应该先解决"架构形式"还是"权力分配"？战略上哪个是因哪个是果？

【第 5 轮】
你把"权力分配"拆成了三件事：预算权、定岗权、战略制定权。我和合伙人又吵了一轮，大家共识是预算权必须收到总部，定岗权下放，战略制定权共创。听起来清晰了，但我越想越觉得这个共识本身站不住——它只解决了"形式上谁签字"，没解决"实质上谁说了算"的问题。再往上推一层：一家公司里真正的权力不来自制度设计，而来自"谁掌握了关键不确定性的解释权"，比如 AI 战略现在就是那个不确定性。这个视角如果成立，我们现在的组织设计是不是从一开始就找错了杠杆？战略上，真正的撬动点到底在哪？

【第 7 轮】
前面六轮我们从中台拆分聊到权力分配，聊到不确定性解释权，最后又聊到创始人心智模型对组织的隐形锚定。我发现我自己也在一层层往上爬，每爬一层都觉得上一层是个伪问题。现在我面前浮现出一个更终极的困惑：我作为创始人，这家公司到底是我的延伸，还是一个应该独立于我而存在的物种？如果是前者，我所有的组织设计其实都是在给自己的决策偏好建基础设施；如果是后者，我现在要做的第一件事是把自己从某些关键位置上抽离出去。这个命题听上去像哲学题，但我觉得它决定了接下来三年公司怎么走。你从战略视角怎么看？我到底该往哪一侧压这个砝码？
```

**四维判定**:
- A/C: **A**（从组织结构 → 权力分配 → 不确定性解释权 → 创始人心智，越聊越抽象）
- M/V: **V**（每轮都重述前情 + 铺垫新判断，典型 Deck 式叙事）
- D/L: **D**（"你从战略视角怎么看""真正的撬动点到底在哪"，指令口吻）
- X/E: **X**（每轮结尾都是方向性发问，持续在扩张可能空间）
- **最终代码**: AVDX

**为什么典型**:
- vs AVDE：AVDE 多轮会一轮比一轮更具体（架构图→岗位卡→JD），AVDX 反过来一轮比一轮更抽象。
- vs AVLX：AVLX 多轮会保持商量口吻且逐渐收敛到情绪/关系问题；AVDX 强势且不断升维，最后推到"公司是不是应该独立于我"的哲学命题，仍在问"该往哪侧压砝码"。

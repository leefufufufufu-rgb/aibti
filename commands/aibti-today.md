# /aibti-today

生成今日 AIBTI 快照，了解今天你和 AI 的协作风格。

## 执行步骤

1. 加载 `aibti` Skill
2. 只扫描今天 (UTC+8) 的 `~/.claude/projects/**/*.jsonl`
3. 按 Skill 逻辑打标签
4. 输出**简化版**报告（无需长篇幅诊断，只要：今日主人格 + 四维 + 3 条代表 prompt）
